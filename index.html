Want to consider switching degrees? Already completed a degree and want to pick up a second one?

Kevin Bacon uses the groscanot API to identify degrees with at least one overlapping course with the degree selected. Degrees are sorted by the number of extra courses needed to get the degree.

Courses that would fall into the 'extra' data field are not shown so there maybe additional courses you needed.
